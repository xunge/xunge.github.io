## windows

### 设备序列号

```
wmic bios get serialnumber
```

### 硬盘序列号

```
wmic diskdrive get serialnumber
```

### 操作系统安装时间

```
systeminfo
```

### IP地址 MAC地址

```
ipconfig /all
```

## Linux

### 设备序列号

```
sudo dmidecode -s system-serial-number
```

### 硬盘序列号

```
sudo lsblk
sudo lsblk -o name,type,serial /dev/nvme0n1
```

### 操作系统安装时间

```
df -P /
sudo dumpe2fs /dev/mapper/ubuntu--vg-root | grep 'Filesystem created:'
```

### IP地址 MAC地址

```
ifconfig
```
